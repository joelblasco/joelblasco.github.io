Labpsitec VR Park

I worked as a freelance for this research lab from Valencia University, this group actively does research on VR medical and clinical applications. They wanted to be able to continue research on a project called Emotions Park, a city park with different settings that aim to change how you feel. Since the project was initially created more than 25 years ago, actual HMDs could not run the game in any way, so I created it from scratch taking into account the old one and the advice from the researchers. The game is now available for Oculus Rift and Oculus Quest. I also modified other projects according to their needs.

  • Oculus VR with Unity.
  • Same environment with different settings.
  • Porting and upgrading older projects.
  • On-site work.

Timerlabs

TimerLabs is a personal project which consists of a software utility for streamers, a chrono-down tool that lets you or your moderators to add time on the go or under certain events provided by Twitch API like new followers, subs, bit donations... creating a .txt file  that can be readed by the streaming platforms like OBS and Streamlabs to share the remaining time on screen.

  • Twitch API
  • Native app launcher for updates.
  • Personal project
